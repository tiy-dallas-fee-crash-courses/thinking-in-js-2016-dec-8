Thinking in JavaScript

In this talk we're going to learn a little bit about thinking in JavaScript. Each file is a progression of this app where we learn one new thing about JavaScript as we go along. To move to the next version of our adventure, just change the file name below.

  1. Telling people something
  2. Asking a question
  3. Checking something
  4. Keep doing the same thing until you get what you want
  5. Organize your code with functions
  6. You can jump from function to function
  7. You can combine as many functions together as you like and extend the story (or solve problems)
